# RemixWarp 文档

[RemixWarp](https://remixwarp.pages.dev/) 的技术文档 - 一个为开发者设计的 Scratch 环境修改版。

## 什么是 RemixWarp？

RemixWarp 是基于 TurboWarp 构建的 Scratch 修改版，专为开发者和高级用户设计：

- **增强型虚拟机** - 带有开发者功能的修改版虚拟机
- **无云存储** - 仅本地执行
- **JavaScript 集成** - 增强的扩展能力
- **开发者工具** - 技术文档和 API
- **修改的界面** - 使用 "Misty" 作为默认角色
